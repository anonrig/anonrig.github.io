About

Portfolio

Indiringo

Indiringo started it's digital life in March 2016 as a place for consumers to freely share deal tips, voucher codes and other consumer information. The purpose of this site is to access to deals, vouchers and freebies easily without any marketing material. Developed using Angular.js with Node.js backend.

Tuşla

Tuşla is a mobile application with more than 330K users that aims to bring the communication experience between brands and customers to the mobile platform. The biggest advantage of all is that Tuşla does not require any integration and / or infrastructural development to use and benefit from any of these features! Developed using Ionic Framework with Node.js backend.

Düşle Beşiktaş

Düşle Beşiktaş is a site led by Beşiktaş city hall. The aim of this site is to build a community in Beşiktaş district that comes up with different ideas to improve Beşiktaş. Developed using Angular.js with Node.js backend.

Izci

Wedding Planner - Ideas
